# Yapz Product Design

## Positioning

Yapz is a browser-based game community and squad chat platform. It uses a server/channel model inspired by modern gaming communication tools: users log in, create or join a server, chat in text channels, and enter voice channels for team play.

## Core Users

- Players who want a lightweight web alternative to native voice tools.
- Small gaming communities that need persistent channels and chat history.
- Team captains who need a simple space for announcements, chat, and voice rooms.

## Main Flows

1. Register or log in.
2. Create a server.
3. Use the default text channel to chat.
4. Join the default voice channel to coordinate.
5. Add more text or voice channels as the server grows.

## First Version

- Email or username password login.
- Server list and active server view.
- Text and voice channel lists.
- Realtime text messages.
- Voice channel join, leave, mute state, and WebRTC signaling transport.
- Online member list.

## Out of Scope for V1

- Screen sharing.
- Mobile native apps.
- Payment.
- Public discovery marketplace.
- Complex role permission editor.
