# Yapz UI Spec

## Layout

The first screen after login is the app workspace.

- Left rail: server icons and create server action.
- Channel sidebar: server name, description, text channels, voice channels, user controls.
- Main panel: current text chat or voice room.
- Right sidebar: server members and roles.

## Visual Direction

- Dark utilitarian game-tool interface.
- Dominant surfaces: near-black, panel charcoal, neutral borders.
- Accents: mint for online/primary actions, coral for destructive actions, amber for secondary emphasis.
- No marketing hero page in the app path.

## Components

- Icon buttons use lucide icons.
- Server icons are compact rounded squares with initials.
- Channel rows use `#` for text and speaker icons for voice.
- Login screen is a compact auth panel with tabs.
- Empty states are direct and action-oriented.

## Responsive Rule

V1 targets desktop first with a minimum practical width around 980px. Mobile support should later collapse sidebars into drawers.
